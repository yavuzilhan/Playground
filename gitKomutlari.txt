** Eklenen yeni dosyaları branch'e eklemek
git add -A 

** Yapılan değişikliklerin geçerli kılınması
git commit -m "Mesaj"

** Local'deki değişikliklerin sunucuya gönderilmesi
git push

** Sunucu ile Local arasındaki farkların kontrolü
git fetch

** Sunucudaki değişiklikleri Local'e çekmek için
git pull
